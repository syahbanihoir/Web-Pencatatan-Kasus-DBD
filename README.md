# Web Sistem Informasi Pencatatan Kasus DBD

Sistem Pencatatan Kasus Demam Berdarah (DBD) berbasis PHP, HTML, CSS, dan MySQL adalah aplikasi web yang dirancang untuk membantu petugas atau instansi terkait dalam mencatat, mengelola, dan memantau laporan kasus DBD secara digital. Sistem ini memudahkan pelaporan data pasien, lokasi kasus. Dengan tampilan antarmuka yang sederhana dan responsif, pengguna dapat dengan cepat memasukkan data dan menghasilkan laporan berbasis wilayah maupun waktu untuk keperluan pemantauan dan pengambilan keputusan yang lebih cepat dan akurat.
